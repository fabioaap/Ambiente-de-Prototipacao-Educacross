# Página 03 - Banco de Questões Revisão

Esta pasta contém a página de gerenciamento do banco de questões com filtros e pesquisa.

## Arquivos

- `index.html` - Estrutura HTML da página
- `script.js` - Lógica JavaScript (filtros, pesquisa, toast)
- `styles.css` - Estilos específicos da página

## Funcionalidades Críticas

- Stats-bar: Exibe badges (Quiz: 15, IA: 5, Humano: 10)
- Toast: Lê 'toastPendente' do localStorage e exibe
- Filtros: Disciplina, Origem, Pesquisa por texto
- Tabela: Renderização de questões com paginação

## Assets

Assets compartilhados estão em `../../assets/`
