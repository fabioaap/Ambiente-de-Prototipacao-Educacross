# Protótipo Jornada — Educacross

Projeto mínimo para colar telas do fluxo e iterar em um protótipo React.

Como rodar (Windows PowerShell):

```powershell
cd "./prototype-react"
npm install
npm run dev
```

Isso iniciará o Vite em `http://localhost:5173`.
