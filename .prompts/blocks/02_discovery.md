# Bloco 02 — Discovery

## Diretrizes de descoberta
- Só ative discovery se houver incerteza técnica relevante.
- Escreva Mini-OST, defina hipótese, métrica-alvo, experimento, rollback.
- Use feature flag com TTL/owner/kill-switch, canário e ADR curto.
- Sem evidência, não promover.

## Fontes
- Priorize docs oficiais, guias, notas de versão. Cite 2-3 links no PR/ADR.
