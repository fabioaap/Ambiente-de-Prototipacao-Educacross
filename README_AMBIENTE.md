# Ambiente de Prototipação - Estrutura de Arquivos

Esta pasta contém a página sobre o Ambiente de Prototipação da Educacross, organizada em arquivos separados seguindo as melhores práticas de desenvolvimento web.

## Estrutura:
- `index.html` - Estrutura HTML
- `base.css` - Tokens do Design System (shadcn/ui)
- `styles.css` - Estilos específicos da página
- `script.js` - Interatividade e comportamento
